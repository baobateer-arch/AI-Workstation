# AI 工作站首页设计规范 v0.4

## 概述

首页定位为 **AI 工作决策中心**，提供一站式工作状态总览，帮助用户快速了解当前工作进展和需要处理的事项。

## 画布规格

- 尺寸：1072 × 1448 像素
- 色彩模式：灰度（L）
- 背景：白色 (255)
- 前景：黑色 (0)
- 灰色：110（中灰）、220（浅灰）

## 区域详细规范

### A. 顶部（高度：120px）

- 左侧：标题 "AI 工作站"
- 右侧：日期时间（灰色小字）
- 底部分隔线：4px 黑色

### B. 今日目标（高度：180px）

- 标题："今日目标"
- 三列数据：
  - 完成项目：X/Y（大数字 + 小标签）
  - 预计收入：¥XXX（大数字 + 小标签）
  - 平均耗时：XX分（大数字 + 小标签）

### C. 当前项目（高度：200px）

- 标题："当前项目"
- 项目名称：大号文字
- 三列数据：
  - 已用时间：XX分
  - 目标时间：XX分
  - AI成本：¥X.XX
- 状态标签：运行中/需要授权/等待回复/任务完成/运行异常/空闲

### D. AI 资源（高度：280px）

- 标题："AI 资源"
- 卡片布局（2列）：
  - Codex：进度条 + 剩余时间
  - DeepSeek：余额 + 状态
  - MiMo：余额 + 状态

### E. Agent 状态（高度：100px）

- 标题："Agent 状态"
- 两列数据：
  - 运行中：X（大数字）
  - 需要处理：X（大数字，需要处理时加粗边框）

### F. 今日收入（高度：180px）

- 标题："今日收入"
- 大号金额：¥X,XXX.XX（超大字体，居中）

### G. AI 建议（高度：280px）

- 标题："AI 建议"
- 建议列表（最多5条）：
  - 编号 + 建议内容
  - 优先级通过边框粗细区分

## 状态显示规范

所有状态使用中文，不使用英文：

| 状态 | 显示文字 |
|------|----------|
| 运行中 | 运行中 |
| 需要授权 | 需要授权 |
| 等待回复 | 等待回复 |
| 任务完成 | 任务完成 |
| 运行异常 | 运行异常 |
| 空闲 | 空闲 |

## 字体规范

- 标题：52px
- 区域标题：36px
- 大数字：48-56px
- 普通文字：28-30px
- 小字：20-22px

## 视觉规则

1. 不使用颜色区分状态
2. 重要信息使用粗边框
3. 紧急信息使用黑底白字
4. 数字尽量大，文字尽量精简
5. 卡片之间保持 16-20px 间距
