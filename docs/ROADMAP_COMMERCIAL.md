# AI Workstation｜商业化路线

这份路线围绕产品体验、交付效率和商业服务展开，不改变当前 Agent 状态检测和监控逻辑的基础边界。

## v0.1｜当前开源版本

定位：可运行的本地 AI 工作状态仪表盘。

- 支持 Claude Code、OpenAI Codex、MiMo Code
- 展示 Agent 状态、Codex 额度、DeepSeek / MiMo 余额
- 支持 Kindle e-ink dashboard
- 提供 Windows 本地运行方式和基础文档
- 通过 Setup Service 提供安装、配置和工作流咨询

## v0.2｜用户体验增强

定位：让更多用户可以更轻松地完成安装和日常使用。

- 优化首次使用引导和产品文档
- 增加设备分辨率与展示布局的适配说明
- 提供更清晰的错误提示、状态解释和恢复建议
- 完善安装服务的交付清单和验收流程
- 增加公开 Demo、案例截图和客户反馈展示

## v1.0｜商业版本

定位：面向个人专业用户、团队和企业的完整产品形态。

- 一键部署：降低安装、更新和环境配置门槛
- Web 控制台：提供更完整的工作流配置和状态查看入口
- 多设备支持：支持多个 Kindle 或其他状态展示设备
- 企业管理：提供团队成员、设备、权限和交付管理能力
