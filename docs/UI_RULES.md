# UI 设计规范

## 画布

- 尺寸：1072 × 1448 像素
- 模式：灰度 (L)
- 背景：白色 (255)
- 前景：黑色 (0)

## 字体规范

| 用途 | 字号 |
|------|------|
| 标题 | 52px |
| 状态大字 | 64px |
| 区域标题 | 36px |
| 大数字 | 48px |
| 普通文字 | 28-30px |
| 小字 | 20-22px |

## 布局规则

1. 边距：48px
2. 卡片间距：16-20px
3. 分隔线：2-4px
4. 圆角：16px

## 视觉规则

1. 白底黑字
2. 大数字突出
3. 高留白
4. 不使用颜色
5. 不使用 emoji 字体
6. 中文优先
7. 无重叠
8. 无裁切

## 状态显示

使用符号区分状态（非颜色）：
- ✓ 正常状态
- ⚠ 警告状态
- ✗ 停止状态

符号后跟中文说明。
