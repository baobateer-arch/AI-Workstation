# Personal observability for AI coding agents

## Body

As developers adopt multiple coding agents, a new operational problem appears: the work is distributed, but attention is still human.

One agent is running. Another is waiting for approval. A third has stopped. Quota may be close to a limit. None of these signals is difficult to find—but finding them repeatedly creates costly context switching.

AI Workstation is my attempt to build a small observability layer for personal AI workflows. It turns a Kindle into an always-visible command center for Claude Code, OpenAI Codex, and MiMo Code.

The design principle is simple: do not reproduce every log. Surface only the states that change human action.

Demo: https://baobateer-arch.github.io/AI-Workstation/
