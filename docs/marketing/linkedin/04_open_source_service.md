# Productizing open source through accountable service

## Body

Open-source users can install software themselves. Customers often pay for a different outcome: speed, confidence, adaptation, and accountability.

AI Workstation remains open source, while the commercial offer focuses on delivery:

- Personal Setup for installation and Kindle configuration;
- Pro AI Workflow for agent orchestration, automation, and custom dashboards;
- Enterprise for team and multi-device scenarios.

This model does not hide the code. It makes the service boundary explicit. A customer knows what will be delivered, what they need to provide, and how success will be verified.

For niche developer tools, service can be the shortest path from interest to a real working outcome.
