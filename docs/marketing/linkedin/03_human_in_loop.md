# Designing the human-in-the-loop moment

## Body

AI-agent productivity is often measured by how much work can be automated. A more practical measure is how quickly the system asks for human judgment at the right moment.

An approval request that sits unseen for ten minutes turns a fast agent into a slow workflow. That is why `WAITING_APPROVAL` is a first-class state in AI Workstation.

The dashboard does not try to automate every decision. It creates a clearer contract between agent and operator:

- keep working when the agent is running;
- intervene when approval is required;
- inspect when work has stopped.

The quality of the handoff matters as much as model performance.
