# Calm technology for an AI-heavy workplace

## Body

Most workplace software competes for attention. AI agents create even more prompts, windows, and status checks.

I wanted to explore the opposite approach: a display that communicates only when action is needed.

AI Workstation uses a Kindle e-ink screen to show durable agent states such as RUNNING, IDLE, WAITING_APPROVAL, and STOPPED. When everything is normal, the screen can be ignored. When human input is required, the change is visible without another notification channel.

E-ink is not only a hardware choice. Its constraints encourage better information design: fewer metrics, stronger hierarchy, and less motion.
