# 我把旧 Kindle 改造成 AI 程序员控制台

## 正文

家里闲置的 Kindle 还能做什么？

我把它改成了一块 AI Workstation：放在显示器旁边，持续显示 Claude Code、OpenAI Codex、MiMo Code 的工作状态。

以前要在多个终端和窗口之间来回切换，才能确认 Agent 有没有跑完、是不是在等授权、额度还剩多少。现在抬眼就能看到：

- 哪个 Agent 正在 RUNNING
- 哪个 Agent 处于 IDLE
- 哪个任务 WAITING_APPROVAL
- Codex 5H / 7D quota 和其他资源余额

电子墨水屏的好处是低功耗、低干扰，像一块安静的 AI 工作状态牌。它不会再增加一个闪烁的窗口，却能一直提醒我下一步该关注什么。

如果你也有闲置 Kindle，或者正在同时使用多个 AI 编程 Agent，可以来聊聊你的工作流。
