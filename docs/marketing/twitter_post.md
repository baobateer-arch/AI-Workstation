# Turn your Kindle into an AI Agent Command Center

## Body

I repurposed an old Kindle into a low-power command center for AI coding agents.

AI Workstation shows Claude Code, OpenAI Codex, and MiMo Code status at a glance:

RUNNING · IDLE · WAITING_APPROVAL · STOPPED

It also tracks Codex 5H / 7D quota, DeepSeek balance, and MiMo balance.

One quiet e-ink screen. Less tab switching. More focus.
