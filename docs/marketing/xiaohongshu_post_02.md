# 3 个 AI Agent 同时工作，我做了一个电子墨水监控屏

## 正文

当 3 个 AI Agent 同时工作时，最容易被忽略的不是代码，而是“它们现在到底在做什么”。

所以我做了一个 AI Workstation，用 Kindle 电子墨水屏做 AI 程序员控制中心：

1. AI Agent Monitoring：查看 Claude Code、OpenAI Codex、MiMo Code 的 RUNNING / IDLE / WAITING_APPROVAL / STOPPED 状态。
2. Resource Tracking：查看 Codex 5H quota、Codex 7D quota、DeepSeek balance、MiMo balance。
3. E-Ink Dashboard：低功耗、低干扰、Always visible。

它更像桌面上的“AI 工作信号灯”，不是又一个需要不断操作的软件。Agent 在后台跑，我只在需要介入时看一眼屏幕。

这类状态总览特别适合长任务、自动化流程和多 Agent 并行开发。
