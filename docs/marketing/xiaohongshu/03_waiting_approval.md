# AI Agent 最浪费时间的状态：它在等你点确认

## 正文

长任务最尴尬的瞬间，不是报错，而是 Agent 十分钟前就停在“等待授权”，你却以为它还在工作。

AI Workstation 把 `WAITING_APPROVAL` 做成一眼可见的状态，放在桌边 Kindle 上。这样我不需要频繁打开终端检查，只要屏幕出现待授权，就知道该切回去处理。

这个设计没有让 Agent 变聪明，却减少了人与 Agent 协作中最常见的空等。很多效率提升并不来自再加一个模型，而是让关键状态在正确的时刻出现。
