# 多个 AI Agent，不应该只是同时打开三个窗口

## 正文

多 Agent 工作流的关键，不是数量，而是分工和介入点。

哪个 Agent 负责实现？哪个负责审查？什么时候需要人工授权？哪些状态必须持续可见？如果这些没有设计，多开几个 Agent 只会带来更多混乱。

AI Workstation 的 Pro AI Workflow（¥999+）做的是场景梳理、Agent 分工、自动化设置和定制仪表盘。先把“人什么时候介入”定义清楚，再决定屏幕上要显示什么。
