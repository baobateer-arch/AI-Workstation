# 3 个 AI Agent 同时工作，我只看这一块屏

## 正文

Claude Code 在改代码，Codex 在跑另一个任务，MiMo 还在处理工具调用。多 Agent 并行很爽，但也带来一个新问题：我开始不停切窗口确认它们有没有卡住。

于是做了 AI Workstation，把三个 Agent 的关键状态汇总到 Kindle：

- RUNNING：正在工作
- IDLE：进程还在，但暂时空闲
- WAITING_APPROVAL：需要我确认
- STOPPED：已经停止

我不需要实时阅读每一行日志，只需要知道“现在是否需要我”。这也是电子墨水屏特别适合的地方：信息少、对比高、一直亮着却不抢注意力。
