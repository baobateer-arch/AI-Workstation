# AI 工作站产品文档 v0.5

## 产品定位

AI 工作驾驶舱 - 为 Kindle Paperwhite 3 设计的 AI 工作状态总览仪表盘。

## 核心功能

1. **状态总览** - 一眼看清当前工作状态
2. **今日战绩** - 完成项目、收入、效率
3. **项目进度** - 当前项目实时状态
4. **Agent 监控** - 各 Agent 运行情况
5. **资源监控** - AI 服务余额和额度
6. **智能建议** - 根据状态自动生成建议

## 状态定义

| 状态 | 含义 | 触发条件 |
|------|------|----------|
| NORMAL | 可以继续工作 | 无异常 |
| WARNING | 需要处理 Agent | 存在 Agent 需要处理 |
| STOP | 需要立即处理 | Codex 不足或资源不足 |

## 使用场景

- 每日开工查看
- 工作中快速确认状态
- 了解 Agent 运行情况
- 监控 AI 资源余额
