# AI Workstation｜产品概览

## 一句话介绍

AI Workstation 把 Kindle 变成 AI Agent 控制中心，让你不用来回切换窗口，也能随时知道 Claude Code、OpenAI Codex 和 MiMo Code 正在做什么。

## 为什么需要它

当多个 AI Agent 同时工作，真正消耗注意力的往往不是写代码，而是确认状态：任务有没有开始、是否正在等待授权、哪个 Agent 已经停止、额度还能支持多久。信息分散在多个终端和应用里，用户只能反复切换窗口。

AI Workstation 将这些关键状态整理成一张低干扰仪表盘，持续显示在 Kindle 电子墨水屏上。它像桌边的一块“AI 工作信号屏”：抬眼即可判断是否需要介入，其余时间继续专注当前工作。

## 核心能力

- **Claude Code Monitor**：查看运行、空闲、等待授权和停止状态。
- **Codex Monitor**：掌握任务状态、待授权动作以及 Codex 5H / 7D quota。
- **MiMo Monitor**：在多 Agent 并行时持续查看 MiMo Code 状态。
- **AI Resource Tracking**：集中显示 Codex quota、DeepSeek balance 和 MiMo balance。
- **Kindle E-Ink Dashboard**：低功耗、低干扰、Always visible。

## 适合谁

- 同时使用两种以上 AI 编程 Agent 的个人开发者。
- 需要观察长任务、自动化任务和待授权操作的独立开发者。
- 希望把闲置 Kindle 重新利用为生产力设备的人。
- 需要为团队设计 AI 工作流、状态展示或多设备方案的技术负责人。
