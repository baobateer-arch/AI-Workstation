# AI Workstation｜客户 FAQ

## AI Workstation 是什么？

它是一套本地 AI Agent 状态仪表盘，把 Claude Code、OpenAI Codex、MiMo Code 的状态与 AI 资源信息集中显示在 Kindle 电子墨水屏上。

## 它和再开一个监控窗口有什么区别？

Kindle 低功耗、低干扰并且始终可见。你不需要切换应用或让监控窗口占用主显示器，只需抬眼确认是否有任务需要介入。

## 支持哪些状态？

当前展示包括 `RUNNING`、`IDLE`、`WAITING_APPROVAL` 和 `STOPPED`。实际可见信息取决于已配置的 Agent 和本地环境。

## 支持哪些资源信息？

当前产品范围包括 Codex 5H / 7D quota、DeepSeek balance 和 MiMo balance。未配置的可选服务不会阻止基础仪表盘运行。

## 所有 Kindle 都可以使用吗？

不同型号的刷新方式和设备条件不同，购买服务前会先确认型号与可行性。Personal Setup 的目标是基于客户已有设备完成可用交付。

## 是否会上传我的代码？

AI Workstation 以本地运行作为设计方向。沟通和交付时不应发送 API Key、Cookie、Token 或私有代码；仅提供完成配置所需的设备与环境信息。

## Personal Setup 与 Pro AI Workflow 有什么区别？

Personal Setup 解决“安装并可用”；Pro AI Workflow 进一步解决“多个 Agent 如何分工、何时自动化、哪些信息需要展示”。

## 服务价格包含 Kindle 吗？

不包含。服务价格不包含 Kindle 硬件、第三方订阅或客户账号费用。
