# AI Workstation｜Enterprise Service

## 面向团队的 AI Agent 可视化与工作流服务

Enterprise Service 面向已经在团队中使用多个 AI Agent，希望统一观察关键状态、降低人工巡检成本并建立清晰介入机制的客户。

## 适用场景

- 多名开发者同时使用 Claude Code、Codex、MiMo 等工具。
- 团队需要一个共享的 Agent 状态视图。
- 需要把多个 Kindle 或展示设备部署到不同工位。
- 希望定制团队关注的 Agent、资源、任务和提醒信息。
- 需要阶段性交付、培训、验收和后续支持。

## 服务范围

### 1. 场景梳理

盘点团队角色、现有 Agent、任务类型、人工审批点和资源限制，确定真正需要展示的信息。

### 2. 方案设计

设计设备布局、状态层级、信息优先级和异常处理方式，避免把仪表盘做成另一块噪声屏。

### 3. 部署交付

根据确认范围完成多设备配置、定制仪表盘和工作流衔接，并按验收清单交付。

### 4. 支持与迭代

根据服务周期提供问题定位、配置调整和使用建议。第三方服务变化或新增功能另行评估。

## 商务流程

需求访谈 → 方案与报价 → 小范围验证 → 正式部署 → 验收与支持。

Enterprise 按设备数量、定制范围、交付周期和支持要求报价。
