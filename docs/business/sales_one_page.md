# AI Workstation｜产品一页介绍

## 把闲置 Kindle 变成 AI 程序员控制中心

AI Workstation 是一套面向 AI 开发者和小型团队的工作状态展示方案：把 Claude Code、OpenAI Codex、MiMo Code 等 Agent 的状态、待处理事项和资源使用情况，集中显示在一块低功耗 Kindle 电子墨水屏上。

它不是又一个需要频繁打开的聊天窗口，而是一块放在桌边、始终可见的工作信号屏。

## 它解决什么问题

当多个 AI Agent 同时工作时，用户经常遇到：

- 不知道哪个 Agent 正在运行、已经空闲，或正在等待授权。
- 需要在多个窗口和终端之间来回切换，才能确认工作进度。
- 忘记查看 Codex 额度、DeepSeek 余额或 MiMo 余额，导致工作突然中断。
- 想保持桌面安静，却又希望重要状态随时可见。

AI Workstation 将这些信息整理成一张清晰的电子墨水仪表盘，让用户抬眼即可判断下一步。

## 核心能力

### AI Agent Monitoring

支持 Claude Code、OpenAI Codex、MiMo Code，并展示 `RUNNING`、`IDLE`、`WAITING_APPROVAL`、`STOPPED` 等工作状态。

### Resource Tracking

集中查看 Codex 5H quota、Codex 7D quota、DeepSeek balance 和 MiMo balance，帮助用户提前发现资源风险。

### E-Ink Dashboard

利用 Kindle e-ink display 的低功耗和低干扰特点，让 AI 工作状态 Always visible。
